# JobsNow

## Setup and Installation

### Install Dependencies
```bash
yarn
```
### Run the Project http://localhost:5173
```bash
yarn dev
```

### Run Run Tests
```bash
yarn test
```

```plaintext
  src/
  |-- __mocks__           # Stub for handling asset imports in Jest tests (e.g., images, stylesheets)
  |-- assets/             # Icons and images
  |-- components/         # Reusable components
  |-- constants/          # Core API URL and navigation links
  |-- hooks/              # Reusable hooks
  |-- pages/              # React Router page components
  |-- router/             # Routing setup for react-router-dom
  |-- store/              # Redux slices and associated logic
  |-- types/              # TypeScript type definitions
```
<small>Installed React using Vite for its fast setup and development speed.</small>
